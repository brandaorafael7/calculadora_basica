# 🧮 Calculadora Simples em Java

Um aplicativo de console em Java que realiza operações matemáticas básicas. Este projeto foi desenvolvido com o objetivo de praticar os conceitos fundamentais da linguagem, como lógica de programação, estruturas de controle e manipulação de dados via console.

---

## 🚀 Funcionalidades

* **Soma** (`+`)
* **Subtração** (`-`)
* **Multiplicação** (`*`)
* **Divisão** (`/`) com validação para evitar divisão por zero.
* **Interface no console:** Menu interativo e intuitivo para o usuário.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java (JDK 17 ou superior)
* **IDE Recomendada:** IntelliJ IDEA, Eclipse ou VS Code

---
